# 🚀 Easply

Easply is a modern productivity mobile application built with Flutter that helps users effortlessly create job application letters, professional CVs, and manage job-related documents in one place.

---

## ✨ Features

- 📝 Create Cover Letters easily
- 📄 Build Professional CVs (ATS & Creative)
- 📦 Bundle and manage PDF documents
- 🖼️ Image to PDF converter
- ✂️ Image tools (resize, crop, background tools)
- 📁 Document management system
- ⚡ Fast & minimal UI for productivity

---

## 🎯 Purpose

Easply is designed to simplify the job application process by providing all essential tools in one application. It focuses on speed, simplicity, and user experience.

---

## 🛠️ Tech Stack

- Flutter (Frontend)
- Dart
- Provider (State Management)
- Firebase (Authentication)
- SQLite (Local Storage)
- Clean Architecture (Layered structure)
